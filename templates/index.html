RUTAS INICIALES

POST /login

Descripción: Mandar usuario y contraseña verifica existencia y devuelve token JWT.

POST /RegisterUser

Descripción: Se necesita datos del usuario para registrarlo en el sistema.

POST /regDevice SOLO PUEDE ACCEDER LOS DISPOSITIVOS

Descripción: el dispositivo debe mandar informacion necesaria para registrarse en la base de dato y este devuelve la logica para que funcione.

POST /Accionar

Descripción: Manipula un topico del arduino requiere jwt y dato.

GET /Historiador

Descripción: Obtiene historial de datos de un topico requiere jwt.

GET /ListarDispo

Descripción: Lista dispositivos asociados a un usuario requiere jwt.
